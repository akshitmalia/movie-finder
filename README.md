# 🎬 Movie Finder App

A simple and interactive movie search application built using the OMDb API, allowing users to explore movies and view detailed information in a clean UI.

---

🌐 Live Demo: https://akshitmalia.github.io/movie-finder/

## 🚀 Features

* 🔍 Search movies by name
* 🎥 Display movie posters, titles, and release years
* 📄 View detailed information:

  * Release date
  * Runtime
  * Genre
  * Director & Actors
  * Plot summary
  * IMDb rating
* ⌨️ Search using Enter key
* 🔙 Back navigation from details view

---

## 🛠️ Tech Stack

* HTML
* CSS (Bootstrap 5)
* JavaScript (Vanilla JS)
* Axios (API calls)
* OMDb API

## ⚙️ Installation & Setup

```bash
git clone https://github.com/akshitmalia/movie-finder.git
cd movie-finder
```

* Open `index.html` in your browser

---

## 🔗 API Used

* OMDb API: https://www.omdbapi.com/

---

## 📌 How It Works

1. User enters a movie name
2. App fetches data from OMDb API
3. Displays list of matching movies
4. Clicking **Details** shows full movie info
5. User can go back to search results

---

## ⚠️ Limitations

* Requires internet connection (API-based)
* Free API has request limits
* No pagination implemented
